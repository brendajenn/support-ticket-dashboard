# 📊 Support Ticket Dashboard (Power BI)

## 📌 Overview

This project presents a Power BI dashboard analyzing customer support ticket data.  
The goal is to identify trends in issue types, resolution time, and overall support efficiency.

---

## 📈 Key Insights

* Majority of tickets are **resolved successfully**, with a smaller portion pending
* **Login and Payment issues** are among the most frequent problems
* High-priority tickets form a significant portion of total requests
* Certain issue types require **longer resolution time**, indicating areas for improvement
